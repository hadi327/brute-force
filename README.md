# brute-force
A Python-based Brute Force Simulation Tool that demonstrates password strength, search space complexity, and estimated cracking time using cryptographic analysis. Built for cybersecurity education and awareness.
Brute Force Simulation Tool 🔐

A cybersecurity education project that simulates how brute-force attacks work by analyzing password complexity and estimating cracking time based on character sets and length.

Features:
- Password strength analysis
- Search space calculation
- Time-to-crack estimation using guess-rate models
- Character set detection (lowercase, uppercase, digits, symbols)
- Visual strength classification (Weak / Moderate / Strong)

Purpose:
This project is built for learning cybersecurity fundamentals and understanding why strong passwords matter in real-world systems.

Tech Stack:
- Python
- Math-based complexity modeling

Disclaimer:
This tool does NOT perform real attacks. It is strictly a simulation for educational purposes.
